## 技术要点

### Excel 导出
- 使用现有 `xlsx` 库
- 后端生成 JSON 数据，前端转换为 Excel
- 或使用后端直接生成二进制文件 (需要额外依赖)

### 扫码枪支持
- 扫码枪本质是键盘输入设备
- 特征：快速输入 + 回车结尾
- 需要全局监听并过滤正常键盘输入

### 条码打印
- 方案 1: 使用 CSS 打印样式 + 浏览器打印
- 方案 2: 生成 PDF 后打印 (需要额外库)
- 推荐方案 1，简单轻量
