## 已实现功能

### v1.0 (2026-04-11) - 基础版本
- [x] 登录系统 - bcrypt 密码加密
- [x] 商品管理 - 增删改查 (CRUD)
- [x] 入库管理 - 采购入库，自动更新库存
- [x] 出库管理 - 销售出库，库存不足检查
- [x] 数据概览 - 商品数、库存、销售、利润统计

- [x] Products.tsx - 分类字段改为非必填

功能改进:
- [x] 商品管理 - 添加搜索功能 (按名称/分类)
- [x] 入库页面 - 添加操作记录列表
- [x] 出库页面 - 添加操作记录列表
- [x] 数据概览 - 添加刷新按钮
- [x] 后端 - 添加入库/出库记录查询接口

### v1.2 (2026-04-12) - 新增功能
- [x] 修改密码功能 - 侧边栏菜单新增"修改密码"入口，支持旧密码验证
- [x] 入库/出库记录日期筛选 - 使用 DatePicker.RangePicker 按日期范围筛选
- [x] 数据导出 Excel - 入库/出库记录表格支持导出为 .xlsx 文件
- [x] 库存预警 - 数据概览页显示库存低于10件的商品预警卡片
